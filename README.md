<p align="center">
A web-based Restaurant Management System built using Laravel and MySQL.
</p>

<hr>

<h2>📌 About The Project</h2>

<p>
<strong>Food Hub</strong> is a web-based Restaurant Management System developed to streamline restaurant operations and improve customer experience. 
The system provides role-based access for <strong>Admin</strong>, <strong>Registered Users</strong>, and <strong>Guests</strong>.
</p>

<p>
It allows administrators to manage food items, orders, and table bookings, while users can browse the menu, add items to cart, and place orders. Guests can book tables along with selected food items for a specific date.
</p>

<h2>⚙️ Installation Guide</h2>

<ol>
  <li>Clone the repository:
    <pre><code>git clone https://github.com/your-username/food-hub.git</code></pre>
  </li>
  <li>Navigate to the project folder:
    <pre><code>cd food-hub</code></pre>
  </li>
  <li>Install dependencies:
    <pre><code>composer install</code></pre>
  </li>
  <li>Create <code>.env</code> file and configure database credentials</li>
  <li>Generate application key:
    <pre><code>php artisan key:generate</code></pre>
  </li>
  <li>Run migrations:
    <pre><code>php artisan migrate</code></pre>
  </li>
  <li>Start development server:
    <pre><code>php artisan serve</code></pre>
  </li>
</ol>
